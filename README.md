# Hospitality Concierge

AI-powered voice concierge built with [ElevenLabs Conversational AI](https://elevenlabs.io/docs/eleven-agents).

## Overview

A real-time voice agent that handles hospitality inquiries — reservations, local recommendations, guest services — through natural conversation in the browser.

## Tech

- **Agent ID:** `agent_5601kk7pz0jkedbrecfyr760yx3y`
- **Integration:** ElevenLabs embeddable widget (CDN)
- **Audio:** WebRTC (auto-negotiated by widget)
- **Hosting:** GitHub Pages

## Setup

No build step required. The site is a single `index.html` that loads the ElevenLabs widget script from their CDN. Microphone access is requested on conversation start.
